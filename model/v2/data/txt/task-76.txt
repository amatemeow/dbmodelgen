Задача: Создать базу данных для автосервиса, содержащую информацию о автомобилях, услугах и заказах. База данных должна содержать следующие таблицы:
1. Таблица "Автомобили" с полями: ID автомобиля, Марка автомобиля, Модель автомобиля, Год выпуска, Пробег.
2. Таблица "Услуги" с полями: ID услуги, Название услуги, Описание услуги, Стоимость услуги.
3. Таблица "Заказы" с полями: ID заказа, Дата заказа, ID автомобиля, ID услуги, Описание заказа.
