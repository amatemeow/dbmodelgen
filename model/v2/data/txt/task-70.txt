Задача: Создать логическую модель базы данных для системы управления учебным процессом.
Система управления учебным процессом предназначена для организации и контроля учебного процесса в образовательном учреждении. Она позволяет создавать учебные планы, назначать преподавателей, регистрировать студентов, отслеживать успеваемость и оставлять комментарии.
Логическая модель базы данных должна включать следующие элементы:
1. Таблица "Учебные планы" с полями:
   - Идентификатор учебного плана (ID)
   - Название учебного плана
   - Описание учебного плана
   - Дата создания учебного плана
   - Дата завершения учебного плана
2. Таблица "Преподаватели" с полями:
   - Идентификатор преподавателя (ID)
   - Имя преподавателя
   - Фамилия преподавателя
   - Должность преподавателя
3. Таблица "Студенты" с полями:
   - Идентификатор студента (ID)
   - Имя студента
   - Фамилия студента
   - Группа студента
4. Таблица "Успеваемость" с полями:
   - Идентификатор студента (ID)
   - Идентификатор учебного плана (ID)
   - Оценка студента
   - Дата оценки
5. Таблица "Комментарии" с полями:
   - Идентификатор комментария (ID)
   - Идентификатор студента (ID)
   - Автор комментария
   - Текст комментария
   - Дата создания комментария
При создании логической модели базы данных необходимо учесть следующие требования:
1. Каждый учебный план должен иметь уникальное название и описание.
2. Каждый учебный план должен быть связан с преподавателем.
3. Каждый студент должен быть зарегистрирован в системе и иметь уникальные имя и фамилию.
4. Каждый студент должен быть связан с учебным планом и иметь оценку по этому плану.
5. Каждый студент может иметь несколько оценок по разным учебным планам.
6. Каждый студент может иметь несколько комментариев от преподавателей.
