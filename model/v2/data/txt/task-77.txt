Задача: Создать базу данных для школы, содержащую информацию о учениках, учителях и оценках. База данных должна содержать следующие таблицы:
1. Таблица "Ученики" с полями: ID ученика, ФИО ученика, Класс, Адрес проживания, Телефон ученика.
2. Таблица "Учителя" с полями: ID учителя, ФИО учителя, Предмет, Опыт работы.
3. Таблица "Оценки" с полями: ID оценки, ID ученика, ID учителя, Предмет, Оценка.
