Задача: Создать базу данных для ресторана, содержащую информацию о блюдах, заказах и клиентах. База данных должна содержать следующие таблицы:
1. Таблица "Блюда" с полями: ID блюда, Название блюда, Описание блюда, Цена блюда.
2. Таблица "Заказы" с полями: ID заказа, Дата заказа, ID клиента, ID блюда, Количество блюд в заказе.
3. Таблица "Клиенты" с полями: ID клиента, ФИО клиента, Адрес доставки, Телефон клиента.
