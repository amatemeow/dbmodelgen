Задача: Создать базу данных для фитнес-клуба, содержащую информацию о клиентах, тренерах и занятиях. База данных должна содержать следующие таблицы:
1. Таблица "Клиенты" с полями: ID клиента, ФИО клиента, Дата рождения, Адрес проживания, Телефон клиента.
2. Таблица "Тренеры" с полями: ID тренера, ФИО тренера, Опыт работы, Специализация.
3. Таблица "Занятия" с полями: ID занятия, Дата занятия, ID тренера, ID клиента, Описание занятия.
