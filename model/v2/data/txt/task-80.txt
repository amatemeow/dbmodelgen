Задача: Создать базу данных для гостиницы, содержащую информацию о номерах, гостях и бронировании. База данных должна содержать следующие таблицы:
1. Таблица "Номера" с полями: ID номера, Название номера, Категория номера, Стоимость номера.
2. Таблица "Гости" с полями: ID гостя, ФИО гостя, Дата рождения, Адрес проживания, Телефон гостя.
3. Таблица "Бронирование" с полями: ID бронирования, Дата бронирования, ID номера, ID гостя, Статус бронирования.
